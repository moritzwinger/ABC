Detailed Description

An ExpressionList is an order list of (potentially null) expressions. However, more specifically, it's intended to be a rough equivalent to the {a,b,c} notation in C++ Note that this is NOT a target (lvalue) but instead an rvalue So we could assign an ExpressionList into a Variable using a VariableAssignment And this would turn the Variable into a vector-valued Variable However, we cannot assign something to an ExpressionList
