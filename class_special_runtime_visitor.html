◆ checkAstStructure()

template<typename T >
void SpecialRuntimeVisitor::checkAstStructure ( AbstractNode &  astRootNode)

Verifies that the structure of the input and output AST meet the expected form. More precisely,

  • the input AST must consists of a single block with declaration statements,
  • and the output AST of a single block with assignment statements.
    Template Parameters
    TThe expected type of the statements in the block.
    Parameters
    astRootNodeThe root node of the AST to check.

The input and output ASTs are expected to consist of a single block with variable declaration statements and variable assignments, respectively.

void SpecialRuntimeVisitor::executeAst ( AbstractNode &  rootNode)

Executes an input program given as AST. NOTE: As our RuntimeVisitor does not handle Functions yet, the visitor must be called on the function's Block. The missing function signature (input and outputs args) are derived from the input and output AST given in the constructor and the getOutput method, respectively.

Parameters
rootNodeThe root node of the input program.
